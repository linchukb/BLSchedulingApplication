Application Title: Scheduling Application.

Application Purpose: Allows user to interface with a database, adding, updating and deleting appointments and other data from it.

Author: Bogdan Linchuk.
Contact Information: blinchu@student.wgu.edu
Version: 1.0
Date: July 9th 2021

Developed using:
IntelliJ 2021.1.3 (Community Edition)
JDK-11.0.11
JavaFX-SDK-11.0-2.2

How to run the program:
(1) Open the java application.
(2) Login
(3) Add/update/delete customers using customer records screen.
(4) Add/update/delete appointments using scheduling screen.
(5) Access reports using reports screen.

Description of the Customer Schedule report:
The report is contains all the appointments for the chosen customer.

MySQL Connector Driver Version Number: mysql-connector-java-8.0.25
